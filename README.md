# 2024 軟體工程師體驗營 - 個人品牌網站

六角學院 2024 軟體工程師體驗營的個人品牌網站作品

Node.js 版本建議為：v14.18.1 以上...

## 安裝

### 取得專案

```
git clone https://github.com/tim8076/2024-web-camp.git 
```

### 安裝套件

```
npm install 
```

### 運行專案

```
gulp 
```

此指令會自動開啟瀏覽器，並運行專案在 `http://localhost:3000/`


## 資料夾說明

- source -靜態資源放置處
  - images -圖片放置處
  - js -js 檔案放置處
  - scss -scss 檔案放置處 

## 專案技術

- Node.js v14.18.1
- scss
- javascript
- gulp

## 第三方服務
